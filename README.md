# Penjajahku App

An Android application developed using Kotlin as the final project for Dicoding's **Belajar Membuat Aplikasi Android untuk Pemula** course. The application provides historical information about Governor-Generals who ruled in the Dutch East Indies through an interactive list and grid interface.

---

## 📖 Overview

Penjajahku App is an educational Android application designed to introduce users to historical Governor-Generals of the Dutch East Indies. Users can browse historical figures through different layouts and view detailed information for each individual.

This project was developed to practice Android fundamentals, including activity navigation, RecyclerView implementation, intent-based data passing, and user interface development using Android Studio.

---

## ✨ Features

- Display historical Governor-Generals in **List View**
- Display historical Governor-Generals in **Grid View**
- Detail page for each historical figure
- Interactive item selection
- Responsive Android user interface

---

## 🛠️ Tech Stack

- Kotlin
- Android Studio
- RecyclerView
- Intent
- ViewBinding
- XML Layout
